PlasmaPy

An open source community developed Python 3.6+ package for plasma physics in the early stages of development.

What is PlasmaPy?

PlasmaPy intends to be for plasma physics what Astropy is for astronomy - a collection of functionality commonly used and shared between plasma physicists and researchers globally, running within and leveraging the open source scientific Python ecosystem.

The goals of this project are more thoroughly described in our vision statement and an earlier conference poster.

Get PlasmaPy

You can get PlasmaPy from pip via pip install plasmapy. We're not on Conda yet, but we're working on it!

Like most scientific Python packages, PlasmaPy probably runs best on the Anaconda distribution.

PlasmaPy requires Python 3.6 and is not compatible with Python 2.
